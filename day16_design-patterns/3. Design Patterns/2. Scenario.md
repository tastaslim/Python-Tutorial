# Example #

Let's take a duck software, you are asked to design a system to manage the ducks. In the begging, you created a base
class Duck, which has few properties like height, weight, speed and methods like display(), sound(), quack(), swim().

```
 abstract class Duck{
    private height: number;
    private weight: number;
    private speed: number;
    public abstract display(): void;
    public sound(): void;
    public quack(): void;
    public swim(): void;
 }
```
